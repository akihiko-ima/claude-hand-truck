# 要求内容

## 概要

`scripts/setup_calibration.py` のターミナル出力を `rich` ライブラリで TUI 化し、操作ガイドやステータス表示の視認性を向上させる。

## 背景

現在の設定スクリプトは `print()` による平文出力のみで、どのステップにいるか・何をすべきかが分かりにくい。rich を使ったカラー表示・パネル・スピナーにより操作ミスを減らす。

## 実装対象の機能

### 1. 起動ヘッダー

- タイトルをパネルで囲んで表示

### 2. カメラ初期化ステータス

- スピナーで「カメラ起動中...」を表示
- 成功/失敗を色付きメッセージで表示

### 3. キャリブレーション手順ガイド

- 各カメラの操作手順をパネルで表示（何をすべきか一目で分かる）
- 完了時に成功メッセージ

### 4. 相互補正プレビューの案内

- プレビューウィンドウを開く前にキー操作を説明するパネルを表示

### 5. 保存完了メッセージ

- 保存ファイルパスをテーブルで表示

## 受け入れ条件

- [ ] `rich` が `pyproject.toml` の依存に追加されている
- [ ] 起動時にタイトルパネルが表示される
- [ ] カメラ初期化中にスピナーが表示される
- [ ] 各ステップの説明がパネルで表示される
- [ ] 保存完了時にファイルパスが色付きで表示される
- [ ] `python scripts/setup_calibration.py` が正常に起動する

## スコープ外

- OpenCV ウィンドウ側のUI変更
- `src/main.py` への rich 導入
- rich 以外の TUI ライブラリ（Textual 等）の使用

## 参照ドキュメント

- `scripts/setup_calibration.py` - 変更対象
- `pyproject.toml` - 依存追加先
